# **Ders 21 Notları: Değer Döndüren Metotlar**

Selam Dostlar! Geçen ders `void` metotları öğrendik. `void` metotlar "Git, bu işi yap" (örn: ekrana yazı yaz) diyen, ama bize bir sonuç geri vermeyen metotlardı.

### **1. Problem: `void` Metottan Sonuç Alamamak**

```csharp
// Bu metot bize 15 sonucunu VERMEZ, sadece ekrana YAZAR.
static void ToplamaYap(int s1, int s2)
{
    Console.WriteLine(s1 + s2);
}

// HATA! 'sonuc' değişkenine atayacak bir değer yok.
// int sonuc = ToplamaYap(10, 5);
```

### **2. Çözüm: Değer Döndüren Metotlar ve `return`**

Eğer bir metodun bize bir değer (sonuç) vermesini istiyorsak:

1. `void` kelimesini kaldırırız.
2. Yerine, metodun döndüreceği verinin tipini (`int`, `string`, `bool` vb.) yazarız.
3. Metodun içinde, `return` anahtar kelimesi ile sonucu geri yollarız.

**`return` Kelimesinin Görevleri:**

1. Belirtilen değeri (sonucu) metodu çağıran yere geri götürür.
2. Metodu o anda sonlandırır (altındaki kodlar çalışmaz).

### **3. `void` vs. Değer Döndüren Metot Karşılaştırması**

| `void` Metot (Eylem Yapar)       | Değer Döndüren Metot (Sonuç Verir)               |
| -------------------------------- | ------------------------------------------------ |
| `static void Selamla(string ad)` | `static string TamAdAl(string ad, string soyad)` |
| `{ Console.WriteLine(ad); }`     | `{ return ad + " " + soyad; }`                   |
| **Çağırma:**                     | **Çağırma:**                                     |
| `Selamla("Ali");`                | `string isim = TamAdAl("Ali", "Veli");`          |
| (İş biter)                       | (`isim` değişkeni "Ali Veli" değerini alır)      |

E-Tablolar'a aktar

### **4. Örnekler**

**`int` (Sayı) Döndürmek:**

```csharp
static int ToplamaYap(int sayi1, int sayi2)
{
    int toplam = sayi1 + sayi2;
    return toplam;
}

// Çağırma:
int sonuc = ToplamaYap(20, 10);
Console.WriteLine(sonuc); // Çıktı: 30
```

**`bool` (Mantıksal) Döndürmek (Çok Yaygındır!):**

```csharp
static bool YasKontrol(int yas)
{
    // if (yas >= 18) { return true; }
    // else { return false; }

    // Daha kısa yolu:
    return (yas >= 18);
}

// Çağırma:
if (YasKontrol(15) == true)
{
    Console.WriteLine("Giriş yapabilir.");
}
else
{
    Console.WriteLine("Giremez!"); // Bu çalışır.
}
```

**Altın Kural:**

- **`void`:** "Git bu işi **yap**."
- **`return`:** "Bunu hesapla ve sonucu bana **getir**."
