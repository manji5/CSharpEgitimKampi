# **Ders 32 Notları: Interface'ler (Arayüzler)**

Selam Dostlar! Bu derste sınıflarımıza çoklu yetenekler kazandırmayı sağlayan Interface kavramını öğrendik.

### **1. Interface Nedir?**

Interface, bir sınıfın **ne yapabileceğini** (yeteneklerini) belirleyen bir şablondur/sözleşmedir. İçinde sadece metotların ve özelliklerin imzaları bulunur, gövdeleri `{ }` boştur.

- İsimleri standart olarak **`I`** harfiyle başlar (`IOrnek`, `IUcabilir`).
- Bir sınıf **birden fazla** interface'i uygulayabilir (implement edebilir). (Abstract sınıftan en büyük farkı budur).

### **2. Tanımlama ve Uygulama**

```csharp
// Yetenek Tanımı
interface ISavasabilir
{
    void Saldir(); // Gövde yok!
}

// Başka Bir Yetenek
interface IYuruyebilir
{
    void Yuru();
}

// Sınıf Tanımı
// Bir sınıf hem bir şeyden miras alıp, hem de birden çok yeteneğe sahip olabilir.
class SavasciOyuncu : Karakter, ISavasabilir, IYuruyebilir
{
    // Sözleşme gereği bu metotları doldurmak ZORUNDADIR.
    public void Saldir()
    {
        Console.WriteLine("Kılıçla saldırdı!");
    }

    public void Yuru()
    {
        Console.WriteLine("Yürüyor...");
    }
}
```

### **3. Hazır Interface Örneği: `IComparable`**

Kendi oluşturduğumuz nesneleri (Örn: `Ogrenci`) sıralamak (`List.Sort()`) istediğimizde, C#'a bu sıralamanın kuralını (Not'a göre mi, isme göre mi?) söylememiz gerekir. Bunu `IComparable` interface'ini uygulayarak yaparız.

```csharp
class Urun : IComparable<Urun>
{
    public string Ad { get; set; }
    public int Fiyat { get; set; }

    // Kıyaslama Metodu
    public int CompareTo(Urun digerUrun)
    {
        // Fiyata göre sırala (Küçükten büyüğe)
        return this.Fiyat.CompareTo(digerUrun.Fiyat);
    }
}

// Kullanım:
List<Urun> sepet = new List<Urun>();
// ... ürünler ekle ...
sepet.Sort(); // Artık çalışır!
```

### **Abstract vs Interface: Hangisi?**

- Eğer **"O Bir ..."** (Is-A) ilişkisi kuruyorsanız (Kedi bir Hayvandır) -> **Abstract Class / Inheritance**.
- Eğer **"O ... Yapabilir"** (Can-Do) ilişkisi kuruyorsanız (Araba Gidebilir, Kuş Gidebilir) -> **Interface**.
