# **Ders 36 Notları: File ve Directory Sınıfları**

Selam Dostlar! Bu derste verilerimizi kalıcı hale getirmenin ilk adımı olan dosya ve klasör yönetimini öğrendik.

### **1. Hazırlık (`System.IO`)**

Dosya işlemleri için kodumuzun en üstüne şu kütüphaneyi eklemeliyiz:

```csharp
using System.IO;
```

### **2. `Directory` (Klasör) Yönetimi**

Klasörleri oluşturmak, silmek ve kontrol etmek için kullanılır. `static` bir sınıftır (`new` gerekmez).

```csharp
string yol = @"C:\Projelerim\YeniKlasor"; // @ işareti \ karakteri için önemlidir.

// Klasör Var mı?
if (Directory.Exists(yol))
{
    Console.WriteLine("Klasör mevcut.");
    // Klasörü Sil (true: İçi dolu olsa bile sil)
    Directory.Delete(yol, true);
}
else
{
    // Klasör Oluştur
    Directory.CreateDirectory(yol);
    Console.WriteLine("Klasör oluşturuldu.");
}
```

### **3. `File` (Dosya) Yönetimi**

Dosyaları kopyalamak, taşımak ve silmek için kullanılır.

- **Önemli Uyarı:** Dosya işlemlerinde her zaman önce `File.Exists()` ile dosyanın varlığını kontrol edin. Yoksa "File Not Found" hatası alırsınız.

```csharp
string kaynak = @"C:\Veri\belge.txt";
string hedef = @"C:\Yedek\belge_yedek.txt";

if (File.Exists(kaynak))
{
    // Kopyalama (hedef dosya zaten varsa hata verir, üzerine yazmaz)
    File.Copy(kaynak, hedef);

    // Taşıma (veya İsim Değiştirme)
    File.Move(hedef, @"C:\Yedek\yeni_isim.txt");

    // Silme
    File.Delete(kaynak);
}
else
{
    Console.WriteLine("Kaynak dosya bulunamadı!");
}
```

Bu komutlar dosyanın **kendisini** yönetir. İçine yazı yazmak veya okumak için bir sonraki dersi bekleyin!
