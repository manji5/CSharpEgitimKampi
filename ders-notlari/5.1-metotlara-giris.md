# **Ders 20 Notları: Metotlara Giriş ve Metot Tanımlama**

Selam Dostlar! Bu derste, kodlarımızı organize etmenin ve kod tekrarını engellemenin 1 numaralı yolunu, yani **Metotları (Methods)** öğrendik.

### **1. Metot Nedir? Neden Kullanırız?**

- **Metot:** Belli bir işi yapmak için paketlediğimiz ve ona bir isim verdiğimiz, yeniden kullanılabilir kod bloklarıdır.
- **Problem (Kod Tekrarı):**
  ```csharp
  Console.WriteLine("Hoş geldin, Ali");
  // ...
  Console.WriteLine("Hoş geldin, Zeynep");
  ```
  Eğer "Hoş geldin" yerine "Merhaba" yazmak isteseydik, iki yeri de değiştirmemiz gerekirdi.
- **Çözüm (Metot):**
  ```csharp
  // 1. Metodu Tanımla (Reçeteyi yaz)
  static void Selamla(string isim)
  {
      Console.WriteLine($"Merhaba, {isim}"); // Değişiklik için tek bir yer!
  }

  // 2. Metodu Çağır (Yemeği pişir)
  Selamla("Ali");
  Selamla("Zeynep");
  ```

**Faydaları:**

1. **Kod Tekrarını Önler (DRY):** Bir işi 10 kez kopyalamak yerine, metodu 10 kez çağırırsınız.
2. **Okunabilirlik:** `MenuyuGoster();` kodu, 10 satırlık menü kodundan daha okunaklıdır.
3. **Yönetilebilirlik:** Bir hatayı düzeltmek veya değişiklik yapmak için tek bir yere bakarsınız.

### **2. `void` Metotlar (Değer Döndürmeyen)**

`void`, metodun "Bu işi yaparım ama sana bir sonuç (değer) geri vermem" demesidir. Sadece bir eylem gerçekleştirir (Ekrana yazı yazdırmak, listeye eleman eklemek vb.).

### **3. Parametresiz Metotlar (Girdi Almayan)**

Parantezleri `()` boş olan metotlardır. Dışarıdan bir bilgiye ihtiyaç duymazlar.

- **Tanımlama:**
  ```csharp
  static void MenuyuGoster()
  {
      Console.WriteLine("--- Menü ---");
      Console.WriteLine("1. Ekle");
      Console.WriteLine("2. Çıkış");
  }
  ```
- **Çağırma:**
  ```csharp
  MenuyuGoster(); // Ekrana menüyü basar
  ```

### **4. Parametreli Metotlar (Girdi Alan)**

Parantezleri `(veriTipi degiskenAdi)` dolu olan metotlardır. Çalışmak için dışarıdan veriye (argümana) ihtiyaç duyarlar.

- **Parametre:** Metodu tanımlarken belirttiğimiz değişken (örn: `string isim`).
- **Argüman:** Metodu çağırırken gönderdiğimiz gerçek değer (örn: `"Ali"`).
- **Tanımlama:**
  ```csharp
  static void ToplamaYap(int sayi1, int sayi2)
  {
      Console.WriteLine($"Toplam: {sayi1 + sayi2}");
  }
  ```
- **Çağırma:**
  ```csharp
  ToplamaYap(10, 5); // 10 ve 5 argümandır. Çıktı: Toplam: 15
  ToplamaYap(100, 50); // Çıktı: Toplam: 150
  ```

Bir sonraki dersimizde, `void` olmayan, yani bize bir sonuç hesaplayıp **geri veren** metotları göreceğiz.
