# **Ders 39 Notları: Basit Menü Navigasyonu**

Selam Dostlar! Bu derste, klasik `1, 2, 3` yazıp Enter'a basma yöntemini bırakıp, yön tuşlarıyla gezilebilen modern bir menü yapmayı öğrendik.

### **1. Temel Mantık**

İnteraktif bir menü 3 aşamadan oluşur:

1. **Çiz:** Seçenekleri ekrana yaz. Seçili olan satırı (`seciliIndex`) farklı renkte yaz.
2. **Dinle:** Kullanıcının bastığı tuşu `ReadKey(true)` ile al.
3. **Güncelle:** Tuş "Yukarı" ise indeksi azalt, "Aşağı" ise artır. Ekranı temizle ve 1. adıma dön.

### **2. Kritik Komut: `ConsoleKeyInfo`**

Kullanıcının bastığı tuşun sadece harfini değil, hangi tuş olduğunu (Örn: F1, Enter, Esc, Ok Tuşları) anlamamızı sağlar.

```csharp
ConsoleKeyInfo tus = Console.ReadKey(true); // true: Ekrana yazma

if (tus.Key == ConsoleKey.UpArrow) { /* Yukarı çık */ }
if (tus.Key == ConsoleKey.Enter)   { /* Seçimi onayla */ }
```

### **3. Menü Döngüsü Şablonu**

Aşağıdaki şablonu tüm projelerinizde kullanabilirsiniz:

```csharp
string[] menu = { "Başla", "Yardım", "Çıkış" };
int index = 0;

while (true)
{
    Console.Clear(); // Ekranı temizle

    // Menüyü çiz
    for (int i = 0; i < menu.Length; i++)
    {
        if (i == index)
        {
            Console.BackgroundColor = ConsoleColor.White; // Seçili olanın arka planı beyaz
            Console.ForegroundColor = ConsoleColor.Black; // Yazısı siyah
            Console.WriteLine($"> {menu[i]}");
            Console.ResetColor(); // Rengi hemen düzelt!
        }
        else
        {
            Console.WriteLine($"  {menu[i]}");
        }
    }

    // Tuşu kontrol et
    var key = Console.ReadKey(true).Key;

    if (key == ConsoleKey.UpArrow)
    {
        index--;
        if (index < 0) index = menu.Length - 1; // Başa dönme (Loop)
    }
    else if (key == ConsoleKey.DownArrow)
    {
        index++;
        if (index >= menu.Length) index = 0; // Sona dönme (Loop)
    }
    else if (key == ConsoleKey.Enter)
    {
        break; // Döngüden çık, seçimi işle
    }
}
```
