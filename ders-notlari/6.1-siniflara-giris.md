# **Ders 24 Notları: Sınıflara Giriş (Blueprint Kavramı)**

Selam Dostlar! Bu derste, C#'ın kalbi olan **Nesne Yönelimli Programlama (OOP)** konusuna giriş yaptık.

### **1. Sınıf (Class) Nedir? (Blueprint Analojisi)**

Şu ana kadar metotlar (fiiller) yazdık. `Sınıf` ise bir "kavramı" veya "nesneyi" (isim) tanımladığımız **taslaktır (blueprint)**.

- **Sınıf (Class):** `Araba`'nın üretim planıdır. Taslaktır.
- **Nesne (Object):** O plandan üretilen _gerçek_ arabadır (Kapıdaki kırmızı BMW).

Sınıf, bir şeyin özelliklerini (verilerini) ve davranışlarını (metotlarını) tanımlar.

### **2. `field` (Alan) ve `private` (Özel)**

- **`field` (Alan):** Bir sınıfın içinde tanımlanan, o sınıfın verilerini (ham maddesini) tutan _değişkenlerdir_.
- **`private` (Erişim Belirleyici):** Bu alanın, sadece o sınıfın _içinden_ erişilebilir olduğunu söyler. Dışarıdan (`Main` metodu vb.) doğrudan erişilemez.
- **Kapsülleme (Encapsulation):** Bu, veriyi koruma yöntemidir. `private` yaparak, dışarıdan `ogrenci.not = -50` gibi geçersiz bir değer atanmasını engelleriz.

```csharp
class Ogrenci
{
    // C# 'da varsayılan (default) erişim belirleyici 'private'dır.
    private string ad;
    private int not;
}
// Dışarıdan 'ogrenci.ad = "Ali"' yapamayız.
```

### **3. `Property` (Özellik) ve `public` (Genel)**

`private` alana (`field`) dışarıdan güvenli bir şekilde erişmek için "kapılar" oluştururuz. C#'ta bu kapılara **`Property` (Özellik)** denir.

- **`public` (Genel):** Bu özelliğin her yerden erişilebilir olduğunu söyler.
- **`get`:** `private` alandaki veriyi _okumamızı_ sağlar (Dışarıya `return` eder).
- **`set`:** `private` alana _değer atamamızı_ sağlar (Dışarıdan gelen `value`'yu alır).

**Uzun Yöntem (Kurallı):**

```csharp
class Ogrenci
{
    private int not; // 1. Kasa (private field)

    public int Not // 2. Güvenli Kapı (public property)
    {
        get { return not; } // Okumak isteyene 'not'u ver
        set
        {
            if (value >= 0 && value <= 100) // Kural koyduk!
                not = value; // Dışarıdan gelen 'value'yu 'not'a ata
        }
    }
}
```

**Kısa (Otomatik) Yöntem (Kural Yoksa):**
Çoğunlukla bu kullanılır. C# `private` alanı arka planda kendi oluşturur.

```csharp
class Ogrenci
{
    // Hem 'get' hem 'set'i olan public bir özellik
    public string Ad { get; set; }
    public int Numara { get; set; }
}
```

Bu yöntem, `private` bir alanı ve ona erişen `get`/`set` kapılarını otomatik oluşturmanın kısayoludur.
