# **Ders 19 Notları: `List<T>` (Jenerik Koleksiyon)**

Selam Dostlar! Bu derste, `ArrayList`'in tehlikeli dünyasından kurtulup, modern C#'ın en çok kullanılan, en güvenli koleksiyonu olan `List<T>`'ye geçiş yapıyoruz.

### **1. `ArrayList`'in Sorunu Neydi?**

`ArrayList`, içine attığınız her şeyi **`object`** (herhangi bir şey) olarak kabul ediyordu. Bu yüzden:

1. `list.Add(100);`
2. `list.Add("Elma");` (Hata vermezdi)
3. `int sayi = (int)list[1];` (Program burada "Elma"yı sayıya çeviremez ve **çökerdi**).

### **2. `List<T>` (Generic) Nedir?**

`List<T>`, "Jenerik" (Generic) bir koleksiyondur. `<T>` harfi, "Type" (Tip) anlamına gelir.

Bu şu demektir: `List<T>`'yi oluştururken, ona **sadece hangi tipten** veri tutacağını en başta söyleriz ve o da sadece o tipi kabul eder.

- `List<int>` : Sadece `int` (tamsayı) kabul eder.
- `List<string>` : Sadece `string` (metin) kabul eder.

Bunu kullanmak için `using System.Collections.Generic;` gerekir.

### **3. `List<T>` Kullanımı ve Avantajları**

**Avantaj 1: Tip Güvenliği (Type Safety)**

```csharp
List<string> isimler = new List<string>();

isimler.Add("Ali");
isimler.Add("Veli");

// HATA! (Yazarken uyarır, program çökmez!)
// isimler.Add(123);
```

Programın çökmesini daha yazarken engelledik!

**Avantaj 2: Kullanım Kolaylığı (Cast Yok!)**
Veriyi geri alırken C# onun tipini zaten bilir, `(string)` gibi dönüşümlere gerek kalmaz.

```csharp
List<int> notlar = new List<int>();
notlar.Add(100);

int ilkNot = notlar[0]; // Direkt int olarak aldık, (int) gerekmedi!
```

**Avantaj 3: `foreach` Kolaylığı**
Döngüde de tip bellidir, `object` ile uğraşmayız.

```csharp
foreach (string isim in isimler)
{
    Console.WriteLine(isim.ToUpper()); // .ToUpper() metodu direkt gelir!
}
```

### **4. Temel Metotlar**

`ArrayList` ile neredeyse aynıdır:

- `.Add(eleman)`: Listenin sonuna eleman ekler.
- `.Remove(eleman)`: Elemanı bulup siler.
- `.RemoveAt(indeks)`: Belirtilen indeksteki elemanı siler.
- `.Count`: Listenin eleman sayısını verir.
- `.Find(kural)`: Belli bir kurala uyan ilk elemanı bulur.C#
  `// "not" 60'tan büyük mü diye bakar
  int gecenNot = notlar.Find(not => not > 60);`

**Altın Kural:** Modern C#'ta dinamik bir listeye ihtiyacınız varsa, `ArrayList`'i **kullanmayın**. Her zaman **`List<T>`** kullanın.
