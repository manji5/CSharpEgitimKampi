# **Ders 37 Notları: Metin Dosyası Okuma/Yazma**

Selam Dostlar! Bu derste metin (`.txt`) dosyalarının içine veri kaydetmeyi ve okumayı öğrendik.

### **1. `using` Bloğu (Hayat Kurtarıcı)**

Dosya işlemlerinde en önemli kural **"Açtığın dosyayı kapat"** tır. Unutursanız dosya kilitli kalır. `using` bloğu, iş bittiği an dosyayı sizin yerinize **otomatik olarak kapatır ve temizler**.

```csharp
using (IslemYapilanSinif nesne = new IslemYapilanSinif())
{
    // İşlemler buraya...
} // Buradan çıkınca nesne otomatik yok edilir (Dispose).
```

### **2. Dosyaya Yazmak (`StreamWriter`)**

Dosyaya yazı yazmak için kullanılır.

- **Üzerine Yazma (Overwrite):** Dosyayı sıfırlar, sadece yenileri yazar. (Varsayılan).
- **Ekleme (Append):** Eski verilere dokunmaz, en sona ekler. (`true` parametresi).

```csharp
string yol = "veri.txt";

// append: true -> Ekleme modu aktif.
using (StreamWriter yazici = new StreamWriter(yol, append: true))
{
    yazici.WriteLine("Merhaba Dünya!");
    yazici.WriteLine("Bu ikinci satır.");
}
```

### **3. Dosyadan Okumak (`StreamReader`)**

Dosyadaki verileri okumak için kullanılır.

- **Önemli:** Okumadan önce `File.Exists(yol)` ile dosyanın varlığını kontrol edin.

**Satır Satır Okuma (En Yaygın):**

```csharp
if (File.Exists(yol))
{
    using (StreamReader okuyucu = new StreamReader(yol))
    {
        string satir;
        // Satır null olmadığı sürece (dosya bitmediği sürece) dön
        while ((satir = okuyucu.ReadLine()) != null)
        {
            Console.WriteLine(satir);
        }
    }
}
```

**Hepsini Tek Seferde Okuma:**

```csharp
using (StreamReader okuyucu = new StreamReader(yol))
{
    string tumIcerik = okuyucu.ReadToEnd();
    Console.WriteLine(tumIcerik);
}
```
