# **8.Ders Notları: Atama ve Bileşik Atama Operatörleri**

Bu derste, değişkenlerimizin değerlerini daha hızlı ve temiz bir şekilde güncellememizi sağlayan kısayolları öğrendik.

### **1. Temel Atama Operatörü (`=`)**

Bu operatör bir soru sormaz, **emir verir**. Sağdaki değeri alır ve soldaki değişkenin içine atar.

```csharp
int yas = 30; // 30 değerini 'yas' değişkenine ata.
```

### **2. Bileşik Atama Operatörleri (Kısayollar)**

Genellikle bir değişkenin değerini, kendi mevcut değerini kullanarak güncellemek isteriz (Örn: `skor = skor + 10`). Bileşik atama operatörleri bu işlemi kısaltır.

**Ana Kural:** `değişken [operatör]= değer;` ifadesi, `değişken = değişken [operatör] değer;` ifadesiyle aynıdır.

| Operatör | Uzun Hali | Kısa Hali (Bileşik) | Anlamı |
| --- | --- | --- | --- |
| `+=` | `x = x + 5;` | `x += 5;` | Topla ve Ata |
| `-=` | `x = x - 5;` | `x -= 5;` | Çıkar ve Ata |
| `*=` | `x = x * 5;` | `x *= 5;` | Çarp ve Ata |
| `/=` | `x = x / 5;` | `x /= 5;` | Böl ve Ata |

**Sayısal Örnek:**

```csharp
int puan = 100;

puan += 20; // puan = 120 (100 + 20)
puan -= 10; // puan = 110 (120 - 10)
puan *= 2;  // puan = 220 (110 * 2)
puan /= 4;  // puan = 55  (220 / 4)

Console.WriteLine($"Son Puan: {puan}"); // Çıktı: 55
```

### **3. `+=` Operatörünün Metinlerle (`string`) Kullanımı**

`+=` operatörü, sayıları topladığı gibi metinleri de birleştirir.

```csharp
string yazi = "C#";
yazi += " öğrenmek"; // yazi = yazi + " öğrenmek";
yazi += " çok keyifli!"; // yazi = yazi + " çok keyifli!";

Console.WriteLine(yazi); // Çıktı: C# öğrenmek çok keyifli!
```
