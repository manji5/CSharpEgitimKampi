# **Ders 17 Notları: `foreach` Döngüsü**

Selam Dostlar! Bu derste, dizilerin içindeki elemanları gezmenin en kolay ve en şık yolunu öğrendik.

### **1. Problem: `for` Döngüsünün Zorluğu**

Bir dizinin elemanlarını `for` ile gezmek istediğimizde, indeksleri takip etmek (`i`), dizinin boyutunu bilmek (`i < 4`) ve indeks ile elemana ulaşmak (`dizi[i]`) zorundayız.

```csharp
int[] sayilar = { 5, 10, 15 };
for (int i = 0; i < 3; i++)
{
    Console.WriteLine(sayilar[i]);
}
```

### **2. Çözüm: `foreach` Döngüsü ("Her Biri İçin")**

`foreach`, diziler ve koleksiyonlar için özelleşmiş bir döngüdür. İndekslerle, boyutlarla uğraşmaz. Sadece dizideki her elemanı sırayla size getirir.

**Sözdizimi:**

```csharp
foreach (veriTipi geçiciDeğişkenAdı in dizininAdı)
{
    // O anki eleman 'geçiciDeğişkenAdı' içindedir.
    // Onu kullan.
}
```

**Örnek 1: Tek Boyutlu Dizi**

```csharp
int[] sayilar = { 5, 10, 15 };

// 'sayilar' dizisindeki her bir 'sayi' için:
foreach (int sayi in sayilar)
{
    Console.WriteLine(sayi);
}
// Çıktı: 5, 10, 15
```

**Örnek 2: İki Boyutlu Dizi**`foreach`, iç içe `for` döngüsü yazma zahmetinden bizi kurtarır.

```csharp
string[,] isimler = {
    { "Ali", "Veli" },
    { "Ayşe", "Fatma" }
};

// 'isimler' matrisindeki her bir 'isim' için:
foreach (string isim in isimler)
{
    Console.WriteLine(isim);
}
// Çıktı: Ali, Veli, Ayşe, Fatma
```

### **3. `for` mu, `foreach` mi?**

- **`for` Kullan:** Eğer elemanın sırasına (indeksine) ihtiyacın varsa veya elemanın değerini **değiştirmek** istiyorsan (`dizi[i] = 100;` gibi).
- **`foreach` Kullan:** Eğer tek amacın dizideki tüm elemanları baştan sona **okumaksa** ve indeksin bir önemi yoksa (Genellikle bu kullanılır).

**Altın Kural:** `foreach` döngüsü **sadece okuma (read-only)** amaçlıdır. Döngü içinde elemanın değerini değiştiremezsiniz.
