# **Ders 30 Notları: Çok Biçimlilik (Polymorphism)**

Selam Dostlar! Bu derste OOP'nin en havalı konusu olan Polimorfizm'i öğrendik. Bir emre farklı nesnelerin farklı tepkiler vermesini sağladık.

### **1. Neden İhtiyacımız Var?**

Miras almada, Ata sınıftaki (`Kisi`) metotlar Çocuk sınıfa (`Ogrenci`) aynen geçer. Ancak bazen Çocuk sınıf, bu metodu kendine göre **değiştirmek (özelleştirmek)** ister.

- `Kisi` -> Sadece Ad Soyad yazar.
- `Ogrenci` -> Ad Soyad'ın yanına **Numara** da eklemek ister.

### **2. `virtual` ve `override` Anahtar Kelimeleri**

Bu değişimi yapmak için iki adım gerekir:

1. **Ata Sınıfta (`virtual`):** Metodun başına `virtual` (sanal) yazılır. Bu, "Çocuklarım bu metodu isterse değiştirebilir" demektir.
2. **Çocuk Sınıfta (`override`):** Metodun başına `override` (ezmek/geçersiz kılmak) yazılır. Bu, "Ben Ata'dan gelen metodu kullanmıyorum, yenisini yazıyorum" demektir.

```csharp
class Hayvan
{
    // Ata izin veriyor:
    public virtual void SesCikar()
    {
        Console.WriteLine("Hayvan sesi...");
    }
}

class Kedi : Hayvan
{
    // Çocuk değiştiriyor:
    public override void SesCikar()
    {
        Console.WriteLine("Miyav!");
    }
}

class Kopek : Hayvan
{
    // Çocuk değiştiriyor:
    public override void SesCikar()
    {
        Console.WriteLine("Hav Hav!");
    }
}
```

### **3. Polimorfizm (Çok Biçimlilik) Nedir?**

Farklı nesnelerin (`Kedi`, `Kopek`), ortak bir referans (`Hayvan`) üzerinden çağrıldığında bile kendi özel davranışlarını sergilemesidir.

```csharp
// Hepsini 'Hayvan' listesine atabiliriz
List<Hayvan> hayvanatBahcesi = new List<Hayvan>();
hayvanatBahcesi.Add(new Kedi());
hayvanatBahcesi.Add(new Kopek());

foreach (Hayvan h in hayvanatBahcesi)
{
    // 'h' değişkeni Hayvan tipinde olsa bile,
    // Kedi ise "Miyav", Köpek ise "Hav Hav" der.
    h.SesCikar();
}
```
