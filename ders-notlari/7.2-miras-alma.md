# **Ders 29 Notları: Miras Alma (Inheritance)**

Selam Dostlar! Bu derste, ortak özellikleri tekrar tekrar yazmaktan kurtulmanın yolunu, Miras Almayı öğrendik.

### **1. Miras Alma (Inheritance) Nedir?**

Farklı sınıflarda ortak bulunan özelliklerin (Ad, Soyad gibi) tek bir "Ata" sınıfta toplanması ve diğer sınıfların bu ata sınıftan türetilmesidir.

- **Base Class (Ata Sınıf):** Özellikleri veren sınıf (`Kisi`).
- **Derived Class (Çocuk Sınıf):** Özellikleri alan sınıf (`Ogrenci`).

### **2. Sözdizimi (`:`)**

C#'ta miras alma iki nokta üst üste `:` ile yapılır.

```csharp
// ATA SINIF
class Kisi
{
    public string Ad { get; set; }
    public string Soyad { get; set; }
}

// ÇOCUK SINIF (Kisi'den miras alır)
class Ogrenci : Kisi
{
    // Ad ve Soyad'ı tekrar yazmana gerek yok.
    // Onlar Kisi'den geldi.
    public int Numara { get; set; }
}
```

**Not:** C#'ta bir sınıf **sadece 1 sınıftan** miras alabilir! (Hem `Kisi`den hem `Robot`tan miras alamaz).

### **3. Constructor ve `base` Anahtar Kelimesi**

Constructor'lar (Yapıcı Metotlar) miras alınmaz. Ancak çocuk sınıf, nesne oluşturulurken aldığı verileri ata sınıfa `base` anahtar kelimesiyle gönderebilir.

```csharp
class Kisi
{
    public string Ad { get; set; }

    public Kisi(string ad) // Ata Yapıcı
    {
        this.Ad = ad;
    }
}

class Ogrenci : Kisi
{
    public int Numara { get; set; }

    // Çocuk Yapıcı
    // 'ad' parametresini alıp 'base(ad)' diyerek Ata'ya paslıyoruz.
    public Ogrenci(string ad, int numara) : base(ad)
    {
        this.Numara = numara;
    }
}

// Kullanım:
Ogrenci o1 = new Ogrenci("Ali", 101);
// 1. "Ali" verisi 'base' aracılığıyla Kisi sınıfına gider ve Ad'ı doldurur.
// 2. 101 verisi Ogrenci sınıfında Numara'yı doldurur.
```
