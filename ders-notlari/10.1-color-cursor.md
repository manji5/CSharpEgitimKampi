# **Ders 38 Notları: Renkler ve İmleç**

Selam Dostlar! Bu derste, sıkıcı siyah konsol ekranını renklendirmeyi ve yönetmeyi öğrendik.

### **1. Renkleri Değiştirmek**

Konsolun renklerini `ConsoleColor` enum'ı ile değiştiririz.

- **`ForegroundColor`**: Yazı rengi.
- **`BackgroundColor`**: Arka plan rengi.
- **`ResetColor()`**: Ayarları varsayılana döndürür. (Bunu kullanmayı unutmayın, yoksa ekranınız renkli kalır!)

```csharp
// Mavi Arka Plan üzerine Beyaz Yazı
Console.BackgroundColor = ConsoleColor.DarkBlue;
Console.ForegroundColor = ConsoleColor.White;

Console.WriteLine(" C# Renkli Konsol Uygulaması ");

// Normale dön
Console.ResetColor();
```

### **2. Ekranı Temizlemek ve İmleci Konumlandırmak**

Konsol bir koordinat sistemidir. Sol üst köşe `(0, 0)` noktasıdır.

- **`Console.Clear()`**: Ekrandaki her şeyi siler.
- **`Console.SetCursorPosition(sutun, satir)`**: İmleci (yanıp sönen çubuğu) belirtilen konuma taşır.
  - **Sütun (Left):** Soldan uzaklık.
  - **Satır (Top):** Yukarıdan uzaklık.

```csharp
Console.Clear(); // Önce temizle

// Ekranın ortasına (tahminen) bir kutu çizelim
Console.SetCursorPosition(10, 5);
Console.WriteLine("----------------");

Console.SetCursorPosition(10, 6);
Console.WriteLine("|  HOŞ GELDİN  |");

Console.SetCursorPosition(10, 7);
Console.WriteLine("----------------");

// İmleci aşağıya gönderelim ki program bitiş yazısı kutuyu bozmasın
Console.SetCursorPosition(0, 10);
```

Bu özellikler sayesinde, akan yazılar yerine sabit arayüzler (Dashboard) tasarlayabilirsiniz.
