# **Ders 26 Notları: Metotlar ve Özellikler**

Selam Dostlar! Geçen ders `new` ile nesne ürettik. Bu derste, o nesneleri nasıl kullandığımızı öğrendik.

### **1. "Nokta" (`.`) Operatörü: Erişim**

Bir nesnenin içindeki `public` (halka açık) özelliklerine (Properties) ve metotlarına (davranışlarına) erişmek için "nokta" kullanırız.

```csharp
Ogrenci ogr1 = new Ogrenci("Ali", 90);

// 'ogr1'in içindeki 'Ad' özelliğine (Property) eriş:
Console.WriteLine(ogr1.Ad); // Çıktı: Ali
```

### **2. Özellikleri (Properties) Kullanma (`get` / `set`)**

Özellikler, nesnenin verilerini (data) temsil eder.

- **Okuma (GET):** Bir özellikten değer okuduğunuzda `get` bloğu çalışır.
  ```csharp
  string adi = ogr1.Ad; // 'Ad' özelliğinin 'get' bloğu çalıştı.
  ```
- **Yazma (SET):** Bir özelliğe değer atadığınızda `set` bloğu çalışır.
  ```csharp
  ogr1.Not = 55; // 'Not' özelliğinin 'set' bloğu çalıştı.
                 // (set içindeki 'value' değişkeni 55 oldu)
  ```

### **3. Nesne Metotları (Instance Methods)**

Bunlar, nesnenin _davranışlarıdır_. `static` kelimesini **kullanmazlar** ve o nesnenin kendi özellikleriyle çalışırlar.

**Sınıfın İçinde Tanımlama (`Ogrenci.cs`):**

```csharp
class Ogrenci
{
    public string Ad { get; set; }
    public int Not { get; set; }

    // (Constructor'lar burada...)

    // 'static' DEĞİL! Bu bir nesne metodudur.
    public void BilgiYazdir()
    {
        // 'this', "BU nesnenin" demektir.
        Console.WriteLine($"Ad: {this.Ad}, Not: {this.Not}");
    }

    public bool GectiMi()
    {
        // Kendi 'Not' özelliğine bakar.
        return this.Not >= 50;
    }
}
```

**Kullanımı (`Main` içinde):**

```csharp
Ogrenci ogr1 = new Ogrenci("Ali", 90);
Ogrenci ogr2 = new Ogrenci("Ayşe", 45);

// 1. Ali için çalışan metot
ogr1.BilgiYazdir(); // Çıktı: Ad: Ali, Not: 90
bool aliGectiMi = ogr1.GectiMi(); // Çıktı: true

// 2. Ayşe için çalışan metot
ogr2.BilgiYazdir(); // Çıktı: Ad: Ayşe, Not: 45
bool ayseGectiMi = ogr2.GectiMi(); // Çıktı: false
```

Metot aynı (`BilgiYazdir`), ama `ogr1` için çağrıldığında Ali'nin verilerini, `ogr2` için çağrıldığında Ayşe'nin verilerini kullanır.
