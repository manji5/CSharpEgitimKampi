# **Ders 16 Notları: Çok Boyutlu Diziler**

Selam Dostlar! Geçen dersimizde tek bir sıradan oluşan (tek boyutlu) dizileri gördük. Bu derste ise verileri bir tablo (Excel gibi) veya matris (satır ve sütun) şeklinde saklamamızı sağlayan çok boyutlu dizileri öğreniyoruz.

### **1. İki Boyutlu Dizi (Matris) Nedir?**

Tek boyutlu diziler `[5]` gibi tek bir indekse sahipken, iki boyutlu diziler bir veriye ulaşmak için iki indeks ister: **`[satir, sutun]`**.

- **Analoji:** Tek boyutlu dizi bir sokaktaki evlerse (`evler[3]`), iki boyutlu dizi bir sinema salonundaki koltuklardır (`salon[5, 12]` -> 5. sıra, 12. koltuk).

### **2. İki Boyutlu Dizi Tanımlama**

Tanımlarken köşeli parantez içine bir virgül (`[,]`) koyarız. Bu, "iki boyut" anlamına gelir.

**Sözdizimi:**

```csharp
// veriTipi[,] diziAdi = new veriTipi[satirSayisi, sutunSayisi];

// 3 satır ve 4 sütundan oluşan bir tamsayı matrisi
int[,] matris = new int[3, 4];
```

- Unutmayın: İndeksler yine **0'dan başlar!**
  - Satır İndeksleri: 0, 1, 2
  - Sütun İndeksleri: 0, 1, 2, 3

### **3. Değer Atama ve Erişme**

Verilere `[satir, sutun]` indeks ikilisiyle erişiriz.

```csharp
int[,] matris = new int[2, 3]; // 2 satır, 3 sütun

// Atama
matris[0, 0] = 10; // 0. satır, 0. sütun (İlk eleman)
matris[0, 1] = 20;
matris[0, 2] = 30;
matris[1, 0] = 40; // 1. satır, 0. sütun
matris[1, 1] = 50;
matris[1, 2] = 60; // 1. satır, 2. sütun (Son eleman)

// Erişme
Console.WriteLine(matris[1, 1]); // Çıktı: 50
```

### **4. Kısa Yoldan Tanımlama (Değerleri Bilerek)**

Dizinin içindeki verileri en başta biliyorsak:

```csharp
// İç içe süslü parantezler kullanırız.
// Her iç parantez bir "satırı" temsil eder.

int[,] sayilar = {
    { 1, 2, 3 },  // 0. Satır
    { 4, 5, 6 }   // 1. Satır
};

Console.WriteLine(sayilar[0, 2]); // Çıktı: 3 (0. satırın 2. indeksi)
Console.WriteLine(sayilar[1, 0]); // Çıktı: 4 (1. satırın 0. indeksi)
```

### **5. Üç ve Daha Fazla Boyut**

Daha fazla boyuta ihtiyacımız varsa, virgül sayısını artırırız:

```csharp
// 3 Boyutlu Dizi (En x Boy x Yükseklik gibi)
int[,,] kup = new int[4, 4, 4];
kup[0, 1, 2] = 100; // 0. katman, 1. satır, 2. sütun
```

- Pratikte en çok tek (`[]`) ve iki boyutlu (`[,]`) diziler kullanılır.
