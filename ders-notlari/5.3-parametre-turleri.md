# **Ders 22 Notları: Parametre Türleri (Değer vs Referans)**

Selam Dostlar! Bu ders, bir metodu çağırdığımızda değişkenlerimizin "arkada" nasıl davrandığını anlamakla ilgili.

### **1. Değer Tipleri (Value Types) - FOTOKOPİ**

- **Nedir?** Veriyi doğrudan içinde tutan basit tipler (`int`, `double`, `bool`, `char`).
- **Davranışı:** Metoda gönderildiğinde, değişkenin bir **kopyası** gider. Metot kopyayı değiştirir, **orijinal değişkeniniz asla değişmez**

```csharp
void Degistir(int kopya) { kopya = 100; }
int x = 10;
Degistir(x);
// x hala 10'dur.
```

### **2. Referans Tipleri (Reference Types) - ADRES**

- **Nedir?** Verinin hafızadaki adresini tutan karmaşık tipler (`List<T>`, `int[]`, `string`, tüm `class`'lar).
- **Davranışı:** Metoda gönderildiğinde, verinin **adresi** gider. Metot bu adresi kullanarak **orijinal veriyi değiştirir.**

```csharp
void Degistir(int[] adres) { adres[0] = 999; }
int[] sayilar = { 10, 20 };
Degistir(sayilar);
// sayilar[0] artık 999'dur! Orijinal liste değişti.
```

---

### **Bu Kuralı Esnetmek: `ref` ve `out`**

Bazen `int` gibi bir değer tipinin de orijinalinin değişmesini isteriz.

### **3. `ref` (Referans ile Yolla)**

- **Amacı:** Değer tiplerini (`int` vb.) metoda "fotokopi" olarak değil, "adres" olarak yollamaya zorlar. Böylece metot, **orijinal değişkeni değiştirebilir.**
- **Kural:** `ref` ile yollanacak değişkenin **mutlaka bir ilk değeri olmalıdır.**
- **Kullanım:** Hem tanımlarken hem çağırırken `ref` yazılır

```csharp
void DegistirRef(ref int orjinal) { orjinal = 999; }
int x = 10; // İlk değer şart!
DegistirRef(ref x);
// x artık 999'dur.
```

### **4. `out` (Çıkış Parametresi)**

- **Amacı:** Bir metodun `return` dışında, **ikinci (veya üçüncü...) bir değer döndürmesini** sağlamak.
- **Kural:** `out` ile yollanacak değişkenin ilk değeri **olmak zorunda değildir**. Metot, o değişkene **içeride bir değer atamak zorundadır.**
- **Kullanım:** "Al bu boş kutuyu (`out`), içini doldur ve bana geri ver" demektir.

**En İyi Örnek - `int.TryParse()`:**`TryParse`, bir metinden sayıya dönüşüm yapmaya çalışır ve bize 2 sonuç verir:

1. (`return`) Başarılı mı? (`bool`)
2. (`out`) Başarılıysa sayı kaç? (`int`)

```csharp
string girdi = "123";
int sonuc; // İlk değer vermedik.

// TryParse'a "sonuc" değişkeninin "out" (çıkış) olduğunu söyledik.
bool basariliMi = int.TryParse(girdi, out sonuc);

// basariliMi == true
// sonuc == 123
```
