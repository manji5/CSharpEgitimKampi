# **Ders 31 Notları: Abstract Sınıflar ve Metotlar**

Selam Dostlar! Bu derste, `virtual` metotların daha katı ve kuralcı versiyonu olan `abstract` yapıları öğrendik.

### **1. Abstract (Soyut) Nedir?**

İçi doldurulmamış, yarım kalmış demektir. Bir sınıfın "Ne yapacağını söyleyip, nasıl yapacağını söylemediği" durumlardır.

- **Abstract Metot:** Gövdesi `{ }` olmayan, sadece imzası olan metottur. "Bu metodu çocuklarım mutlaka kendine göre yazmalı" demektir.
- **Abstract Sınıf:** İçinde abstract metot barındıran sınıftır. Kendisinden **nesne üretilemez** (`new` yapılamaz).

### **2. Neden Kullanırız?**

Belirli bir kural setini (şablonu) çocuk sınıflara **zorunlu kılmak** için kullanırız.

Örneğin: "Her Şeklin alanı hesaplanmalıdır. Ama nasıl hesaplanacağını Kare ve Daire kendisi bilir."

### **3. Sözdizimi ve Kurallar**

```csharp
// 1. 'abstract' sınıf
abstract class Sekil
{
    // Normal özellikler olabilir
    public string Renk { get; set; }

    // 2. 'abstract' metot (Gövdesi YOK, sonu noktalı virgül)
    public abstract double AlanHesapla();

    // Normal metotlar da olabilir
    public void Yazdir() { Console.WriteLine("Ben bir şekilim."); }
}

// 3. Miras Alan Sınıf
class Kare : Sekil
{
    public int Kenar { get; set; }

    // 4. ZORUNLULUK: Abstract metodu 'override' etmek ZORUNDASINIZ.
    // Yoksa program derlenmez (Hata verir).
    public override double AlanHesapla()
    {
        return Kenar * Kenar;
    }
}
```

### **4. `virtual` vs `abstract` Farkı**

| Özellik           | `virtual` Metot                         | `abstract` Metot                               |
| ----------------- | --------------------------------------- | ---------------------------------------------- |
| **Gövdesi { }**   | Vardır (Varsayılan işi yapar).          | **Yoktur.**                                    |
| **Sınıf Türü**    | Normal sınıfta bulunabilir.             | Sadece **Abstract** sınıfta bulunabilir.       |
| **Zorunluluk**    | Çocuk sınıf ezmek zorunda **değildir**. | Çocuk sınıf ezmek (`override`) **ZORUNDADIR**. |
| **Nesne Üretimi** | Sınıfından `new` yapılabilir.           | Sınıfından `new` **yapılamaz**.                |

E-Tablolar'a aktar

**Özet:** `virtual` "İstersen değiştir", `abstract` "Değiştirmek (yazmak) zorundasın" demektir.
