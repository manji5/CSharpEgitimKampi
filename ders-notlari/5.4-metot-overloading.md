# **Ders 23 Notları: Metot Overloading (Aşırı Yükleme)**

Selam Dostlar! Bu derste, kodumuzu daha esnek ve okunabilir kılan "Metot Aşırı Yükleme" konusunu öğrendik.

### **1. Metot Overloading Nedir?**

Overloading (Aşırı Yükleme), aynı isimde (`Topla` gibi) birden fazla metot tanımlamamıza izin veren bir C# özelliğidir.

### **2. Kural: Metot İmzası (Method Signature)**

C#'ın bu metotları karıştırmaması için, **imzalarının** farklı olması gerekir.
Bir metodun imzası = **Metot Adı + Parametre Listesi** (Parametrelerin sayısı, tipi veya sırası).

**ÖNEMLİ:** Metodun **dönüş tipi** (`int`, `void` vb.) **imzanın bir parçası DEĞİLDİR.**

```csharp
// HATA! Bu ikisi aynı imzaya sahiptir, birlikte kullanılamaz.
int Topla(int a, int b);
void Topla(int a, int b);
```

### **3. Overloading Nasıl Yapılır?**

İsim aynı kalmak şartıyla:

**1. Parametre Sayısı Farklı Olabilir**

```csharp
static int Topla(int a, int b) { ... }
static int Topla(int a, int b, int c) { ... }
```

**2. Parametre Tipleri Farklı Olabilir:**

```csharp
static int Topla(int a, int b) { ... }
static double Topla(double a, double b) { ... }
```

**3. Parametre Sırası Farklı Olabilir**

```csharp
static void BilgiVer(string isim, int yas) { ... }
static void BilgiVer(int yas, string isim) { ... }
```

### **4. En İyi Örnek: `Console.WriteLine()`**

Bu özelliği serinin başından beri kullanıyoruz!

```csharp
Console.WriteLine(10); // int overload'u çağırır
Console.WriteLine("Metin"); // string overload'u çağırır
Console.WriteLine(true); // bool overload'u çağırır
```

İsim hep aynı, C# verdiğiniz veriye göre doğru metodu kendi seçiyor.
