# **Ders 25 Notları: Nesne Oluşturma ve Constructor Metotları**

Selam Dostlar! Geçen ders `class` (sınıf) ile bir nesnenin **taslağını (blueprint)** çizdik. Bu derste o taslaktan _gerçek_ bir nesne üretiyoruz.

### **1. `new` Anahtar Kelimesi (Nesne Üretme)**

`class` (taslak) tek başına bir hiçtir. Ondan bir "örnek" (instance) veya "nesne" (object) üretmemiz gerekir. Bu işi `new` anahtar kelimesi yapar.

- **Sınıf (Class):** `Ogrenci` (Bu bir taslaktır).
- **Nesne (Object):** `new Ogrenci()` (Bu, o taslaktan üretilmiş _gerçek_ bir öğrencidir).

```csharp
// Sözdizimi: SinifAdi degiskenAdi = new SinifAdi();

// Artık 'ogr1', hafızada yer kaplayan gerçek bir Ogrenci nesnesidir.
Ogrenci ogr1 = new Ogrenci();
```

### **2. Constructor (Yapıcı Metot) Nedir?**

`new Ogrenci()` yazdığımız anda, `Ogrenci` sınıfının içinde **otomatik olarak çalışan** özel bir metottur.

- **Görevi:** Nesneyi ilk kullanıma hazırlamak, varsayılan değerlerini atamak. (Fabrikadan çıkan arabanın ilk ayarlarının yapılması gibi).
- **Kuralları:**
  1. Adı, sınıf adıyla **birebir aynı** olmalıdır.
  2. **Dönüş tipi yoktur** (`void`, `int` vb. yazılmaz).

**Varsayılan (Default) Constructor:**
Eğer biz hiç constructor yazmazsak, C# bizim için boş bir tane oluşturur:

```csharp
class Ogrenci
{
    // C#'ın otomatik oluşturduğu gizli metot:
    public Ogrenci() { }

    public string Ad { get; set; }
}
```

### **3. Constructor Overloading (Aşırı Yükleme)**

Nesneyi oluştururken (`new` anında) ona ilk verilerini de yollamak isteyebiliriz. Tıpkı metotlar gibi, constructor'lar da aşırı yüklenebilir.

```csharp
class Ogrenci
{
    public string Ad { get; set; }
    public int Not { get; set; }

    // 1. Overload (Parametresiz - Boş)
    public Ogrenci()
    {
        this.Ad = "Girilmedi";
    }

    // 2. Overload (Parametreli)
    // 'this' kelimesi, "bu nesnenin" anlamına gelir.
    public Ogrenci(string ad, int not)
    {
        this.Ad = ad;
        this.Not = not;
    }
}
```

**Kullanımı:**

```csharp
// 1. Boş constructor'ı kullanır:
Ogrenci ogr1 = new Ogrenci(); // ogr1.Ad = "Girilmedi"
ogr1.Ad = "Ali"; // Değeri sonra atadık

// 2. Dolu constructor'ı kullanır (Tek satırda!)
Ogrenci ogr2 = new Ogrenci("Ayşe", 95); // ogr2.Ad = "Ayşe", ogr2.Not = 95
```

Artık nesnelerimizi hem boş hem de dolu olarak, tek satırda oluşturabiliyoruz!
